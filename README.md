# Projets IA réalisés durant la formation Ingénieur IA / Data Scientist OpenClassrooms.

* Machine Learning supervisé,
* apprentissage non supervisé,
* traitement automatique du langage (NLP),
* vision par ordinateur,
* systèmes de recommandation.

## Projets

### Scoring crédit

Modélisation du risque de défaut de paiement à partir de données financières historiques.

Approche :

* analyse exploratoire des données,
* préparation et enrichissement des variables,
* modèles de classification supervisée,
* optimisation et évaluation des performances.

### Segmentation clients e-commerce

Analyse de données transactionnelles afin d'identifier différents profils clients.

Approche :

* consolidation de données multi-sources,
* analyse RFM,
* clustering K-Means et DBSCAN,
* interprétation des segments obtenus.

### Analyse de sentiments NLP

Classification automatique de sentiments à partir de données textuelles.

Approche :

* prétraitement et représentation des textes,
* comparaison de modèles Machine Learning et Deep Learning,
* réseaux LSTM,
* modèle Transformer BERT.

### Segmentation d'images urbaines

Développement d'un modèle de segmentation sémantique appliqué à la compréhension de scènes urbaines.

Approche :

* préparation d'images et de masques,
* réseaux convolutifs,
* architectures U-Net,
* Transfer Learning avec modèles pré-entraînés.

### Labellisation automatique d'images

Développement d'une approche de classification automatique d'images basée sur l'apprentissage non supervisé.

Approche :

* analyse exploratoire d'un jeu d'images,
* prétraitement et amélioration des images,
* extraction de descripteurs visuels SIFT,
* représentation Bag of Visual Words,
* réduction de dimension avec PCA et t-SNE,
* clustering K-Means et évaluation des résultats.

### Système de recommandation

Développement d'un moteur de recommandation hybride basé sur les interactions utilisateurs et les caractéristiques des articles.

Approche :

* filtrage basé contenu,
* embeddings,
* filtrage collaboratif,
* factorisation matricielle,
* exposition du modèle via API.
